<div class="brinza-task-description">
<p>Days of the week are represented as three-letter strings (<tt style="white-space:pre-wrap">"Mon"</tt>, <tt style="white-space:pre-wrap">"Tue"</tt>, <tt style="white-space:pre-wrap">"Wed"</tt>, <tt style="white-space:pre-wrap">"Thu"</tt>, <tt style="white-space:pre-wrap">"Fri"</tt>, <tt style="white-space:pre-wrap">"Sat"</tt>, <tt style="white-space:pre-wrap">"Sun"</tt>).</p>
<p>Write a function <tt style="white-space:pre-wrap">solution</tt> that, given a string S representing the day of the week and an integer K (between 0 and 500, inclusive), returns the day of the week that is K days later.</p>
<p>For example, given S = <tt style="white-space:pre-wrap">"Wed"</tt> and K = 2, the function should return <tt style="white-space:pre-wrap">"Fri"</tt>.</p>
<p>Given S = <tt style="white-space:pre-wrap">"Sat"</tt> and K = 23, the function should return <tt style="white-space:pre-wrap">"Mon"</tt>.</p>
